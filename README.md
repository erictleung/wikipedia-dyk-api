# wikipedia-dyk-api

An API to serve requests for announcements, articles, and facts listed in Wikipedia's "Did you know.."

**Contents**

- [Setup](#setup)
- [Run locally](#run-locally)

## Setup

```bash
python3 -m venv wikidyk

source wikidyk/bin/activate
```

## Run locally

Run development:

```bash
fastapi dev
```

Access the documentation page for this API using http://127.0.0.1:8000/docs.
